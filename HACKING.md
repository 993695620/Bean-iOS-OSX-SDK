# Developer Guide

This file contains instructions that apply to developers only.

# Testing

The Bean SDK contains integration tests that are intended for use with physical Bean hardware. Unit tests are also included to verify logic.

For information on using/running/building these tests, consult [Apple's documentation](https://developer.apple.com/library/ios/documentation/DeveloperTools/Conceptual/testing_with_xcode/testing_4_running_tests/testing_4_running_tests.html#//apple_ref/doc/uid/TP40014132-CH5-SW1).

To run these tests from the command line, consider using [xctool](https://github.com/facebook/xctool).
